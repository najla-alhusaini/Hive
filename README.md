# Hive Care System 🐝

A web-based Beekeeping Management System designed to help beekeepers manage apiaries, monitor hive health, and organize production data efficiently.

## 📌 Project Overview
This system addresses the problem of fragmented and manual beekeeping data by providing a centralized digital platform that simplifies data management and improves productivity.

## 🎯 Features

### 👤 User Management
- Register, login, and logout
- Manage beekeeper accounts (view, activate, deactivate, delete)

### 🐝 Apiaries Management
- Add, view, edit, delete apiaries records
- Search and monitor apiaries across users

### 🩺 Hive Health Records
- Record hive health status
- Identify issues and track hive conditions
- Store inspection notes and treatments

### 📊 System Reports
- Generate reports on platform usage
- View number of users and apiaries
- Display activity logs
- Export reports (PDF / CSV)

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` using a web browser

## 👥 Target Users
- Beekeepers with basic computer skills
